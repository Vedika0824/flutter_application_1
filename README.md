Frontend (Flutter)
#Frontend Requirements
  Flutter SDK (3.10.0)
  VS Code
  Dart
  real device

Setup Instructions
Clone the repository:

git clone https://github.com/Vedika0824/flutter_application_1.git

Install dependencies:
flutter pub get

Run the app on an emulator or device:
flutter run

Configure API IP:
Tap the settings icon in the AppBar.
Enter the backend IP (e.g., 192.168.1.10:3000) for testing.
Android device and system should be connected to the same network(WiFi).

Key Features
Product Management
Add, list, search, and validate products.
Cart System

Bill Generation
Generate bills, view bill history.
